## cooler-game-of-life
This is my java implementation of Conway's Game of Life.

### Features:
- Basic Convay's Game of Life rules

### How to play:
- Run CoolerGameOfLifeApplication class;
- Open http://localhost:9090 in your browser;
- Put some cells on the grid;
- Press "Play"
